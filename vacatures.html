Microsoft Azure/.NET Developer ☆☆☆

De Microsoft Azure cloud biedt een grote verscheidenheid aan services, waarbij de schaalbaarheid niet ter discussie staat. Het op een goede manier inzetten, uitbreiden en koppelen van deze services is geen sinecure.
Voor jou is dat echter een leuke uitdaging! Je hebt ruime ervaring in het ontwikkelen van C#, ASP.NET MVC-applicaties al dan niet op het Azure platform gehost. Uiteraard heb je Microsoft SQL Server gebruikt en doe je de data-modelling geheel zelfstandig. Als je wel eens Azure logic apps en Azure functional apps applicaties hebt ontwikkeld, is dat een pre. DevOps is voor jou geen loze kreet.
Microsoft ontwikkelt in rap tempo een eigen low-code platform onder de vlag van Microsoft Office. Ook deze ontwikkelingen, zoals Microsoft Flow en Microsoft PowerApps, volg jij op de voet.

De volgende punten zijn je dan ook op het lijf geschreven:

  • Een goede basisopleiding is natuurlijk essentieel, maar los daarvan zit model based design in je DNA;
  • Je kunt deze modellen goed vertalen naar werkende applicaties, die full-code of low-code kunnen zijn, een frontend kunnen hebben of als een API functioneren;
  • Je kunt een klantdomein goed vertalen naar het proces- en dataperspectief;
  • Je gereedschap is: C#, ASP.NET MVC ,SQL Server, (mogelijk)Azure Logic apps & Azure Functions en alles wat er via de Cloud beschikbaar is/komt/gaat komen;
  • Je ziet de toekomst van software engineering in het ontwerpen en bouwen van cloudcomponenten.

Geïnteresseerd? Neem contact met ons op!

Microsoft Operations Engineer ☆☆☆

Voor onze klanten ontwikkelen wij online oplossingen. De gebruikers en andere afnemers van deze oplossingen stellen hoge eisen aan de betrouwbaarheid en beschikbaarheid ervan. Onze ontwikkelaars willen gecontroleerd updates kunnen uitvoeren aan de door hen ontwikkelde services. In dit speelveld is de operations engineer een belangrijke speler. Als Microsoft partner maken wij gebruik van de Azure cloud infrastructuur. Jouw rol als Microsoft operations engineer is om deze dynamische omgeving stabiel te krijgen en te houden. Hierbij word je ondersteund door de state of the art cloud tools die Microsoft te bieden heeft, maar ben je ook in staat deze tools door middel van scripting zelf te ontwikkelen.

Je bent een ervaren beheerder van web applicaties en/of cloud services en je herkent je in onderstaande punten of je zou jezelf hierin graag willen ontwikkelen:

  • Ervaring in het beheren van Microsoft Azure cloud oplossingen;
  • Ervaring in het inrichten en opzetten van Microsoft Azure native cloud services, zoals Microsoft Azure App service en Microsoft Azure SQL database;
  • Ervaring in het inrichten en opzetten van Microsoft Azure Virtual machines;
  • Monitorren van Microsoft Azure native cloud services en het instellen van alerts;
  • Het opzetten van een CI/CD straat in Microsoft Azure DevOps;
  • Staan voor het stabiel krijgen en houden van de oplossing voor onze klanten;
  • In staat snel te leren van nieuwe native cloud services die Microsoft zal introduceren.

Geïnteresseerd? Neem contact met ons op!
